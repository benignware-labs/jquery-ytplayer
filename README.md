jquery-ytplayer
==============

> Wrapper for the YouTube-IFrame-API

Usage
-----

Include dependencies

```html
<script src="https://code.jquery.com/jquery-2.2.1.min.js"></script>
<script src="src/jquery.ytplayer.min.js"></script>
```

Provide markup

```html
<div class="ytplayer" data-video-id=""> </div>
```

Initialize Plugin

```js
$('.ytplayer').ytplayer();
```
